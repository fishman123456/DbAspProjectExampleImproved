ТЗ  "Разработка API товары и заказы"
Необходимо разработать web-api приложение, позволяющие работать с заказми и товарами.

Модель БД:
1) Client (клиент):
	- Id: int
	- FirstName: string
	- LastName: string
	- Email: string
	- BirthDate: DateTime
2) Product (товар):
	- Id: int
	- Title: string
	- Description: string
	- Price: decimal
	- Quantity: int
3) Category (категория товара):
	- Id: int
	- Description: string
4) Order (заказ):
	- Id: int
	- Description: string
	- CreatationDate: DateTime
	- IsCompleted: bool
5) OrderProduct:
	- Id: int
	- Quantity: int
Связи:
	Client(1)	-> Order(n)
	Order(1)	-> OrderProduct(n)
	Product(1)	-> OrderProduct(n)
	Category(1)	-> Product(n)

Функции, которые должны выполняться API:

1) Client:
	- зарегистрировать клиента; email должен быть уникальным; клиенту должно быть 18 или более лет;
	- получить список клиентов
	- получить клиента по id
	- редактировать данные клиента
2) Product:
	- добавить товар; price >= 0; quantity >= 0
	- получить список товаров
	- получить товар по id
	- редактировать товар
	- получить товар по id категории
3) Category:
	- добавить категорию
	- реадктировать категорию
	- получить все категории
	- получить категорию по id
	- получить кол-во товаров в заданной по id категории
	- удалить категорию; удалять можно только категорию, в которой отсутствуют товары
4) Order:
	- сформировать новый заказ для заданного по id клиента (у данного клиента не должно быть незавершенных заказов)
	- добавить в заказ товар, заданный по id, при этом проверить, что данного товара достаточно, 
	а после добавления уменьшить кол-во товаров в таблице товаров  (если товар уже существует в заказе, то увеличить кол-во этого товара в заказе)
	- удалить товар, заданный по id из заказа, при этом увеличить кол-во товаров в таблице товаров
	- завершить оформление заказа (с завершенным заказом нельзя добавлять/удалять товары)
	- получить заказ по id
	- получить заказы по id клиента
	- получить список всех заказов
	- редактирование описания заказа (редактировать можно только незавершенный заказ)
	- получить информацию о стоимости товаров в заказе (стоимость каждого товара с учетом кол-ва) и общая стоимость заказа - т.е. чек
